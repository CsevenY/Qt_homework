# 图书管理系统

基于Qt和SQLite开发的图书管理系统，实现了图书信息管理、读者档案管理、借还书流程和统计功能。

## 功能特性

### 核心功能
1. **图书信息录入与维护**
   - 添加、修改、删除图书信息
   - 支持ISBN、书名、作者、出版社、出版日期、分类、价格等信息管理
   - 图书副本数量管理

2. **读者档案管理**
   - 添加、修改、删除读者信息
   - 支持读者编号、姓名、性别、电话、邮箱、地址等信息管理
   - 读者状态管理（正常、暂停、注销）

3. **借还书流程记录**
   - 借书操作：记录借阅日期和应还日期
   - 还书操作：记录归还日期，自动计算逾期罚款
   - 借阅记录查询和筛选

4. **借阅数据统计**
   - 图书总数统计
   - 读者总数统计
   - 总借阅次数统计
   - 当前借出数量统计
   - 逾期数量统计
   - 已归还数量统计

### 技术特性
- **基于SQLite数据库**：轻量级、无需额外配置
- **Model/View架构**：使用QSqlTableModel实现数据模型，QTableView显示数据
- **多条件筛选查询**：支持按ISBN、书名、作者、分类等条件组合查询
- **逾期自动提醒**：定时检查逾期记录，在状态栏显示提醒信息
- **自动布局UI**：使用Qt布局管理器实现响应式界面

## 数据库设计

### 图书表 (books)
- id: 主键
- isbn: ISBN号（唯一）
- title: 书名
- author: 作者
- publisher: 出版社
- publish_date: 出版日期
- category: 分类
- total_copies: 总册数
- available_copies: 可借册数
- price: 价格
- description: 描述

### 读者表 (readers)
- id: 主键
- reader_id: 读者编号（唯一）
- name: 姓名
- gender: 性别
- phone: 电话
- email: 邮箱
- address: 地址
- register_date: 注册日期
- status: 状态

### 借阅记录表 (borrow_records)
- id: 主键
- reader_id: 读者编号（外键）
- book_isbn: 图书ISBN（外键）
- borrow_date: 借阅日期
- due_date: 应还日期
- return_date: 归还日期
- status: 状态（借出/已归还）
- fine_amount: 罚款金额

## 编译和运行

1. 使用Qt Creator打开 `LibraryManagementSystem.pro`
2. 配置Qt版本（建议Qt 6.x）
3. 编译项目
4. 运行程序

## 使用说明

### 图书管理
1. 在"图书管理"标签页中，可以添加、修改、删除图书信息
2. 使用搜索功能可以按ISBN、书名、作者、分类进行筛选
3. 点击表格中的行可以选中并编辑该图书

### 读者管理
1. 在"读者管理"标签页中，可以添加、修改、删除读者信息
2. 使用搜索功能可以按读者编号、姓名、电话、状态进行筛选
3. 点击表格中的行可以选中并编辑该读者

### 借还书管理
1. 在"借还书管理"标签页中，可以办理借书和还书业务
2. 输入读者编号和图书ISBN，设置借阅天数（默认30天），点击"借书"
3. 在表格中选择借阅记录，点击"还书"完成归还
4. 系统会自动计算逾期罚款（每天0.5元）

### 数据统计
1. 在"数据统计"标签页中查看各项统计数据
2. 点击"刷新统计"按钮更新统计数据

### 逾期提醒
- 系统每小时自动检查一次逾期记录
- 发现逾期记录时会在状态栏显示提醒信息
- 逾期记录在表格中会以红色背景高亮显示

## 注意事项

1. 确保数据库目录存在：`E:\Qt_project\Qt_homework\LibraryDB\`
2. 借书前请确保图书有可借副本
3. 还书时会自动计算逾期罚款
4. ISBN和读者编号必须唯一
